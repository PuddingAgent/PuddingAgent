## 3. 回滚方案

| 风险 | 概率 | 回滚 |
|------|------|------|
| SSE 帧类型不兼容 | 低 | 前端降级忽略未知事件类型 |
| 终端代理权限不足 | 中 | 默认禁用 Shell Execution，测试用白名单命令 |
| 数据库 migration 失败 | 低 | 删除 `agent_data/` volume 重新初始化 |
| 内存泄漏（进程残留） | 中 | `ReapAsync` 自动收割 + 定期 GC |

## 5. Playwright 端到端 UI 测试

### 5.1 安装

```bash
cd Source/PuddingPlatformAdmin
pnpm add -D @playwright/test
npx playwright install chromium
```

### 5.2 测试目录

```
Source/PuddingPlatformAdmin/e2e/
├── playwright.config.ts
├── chat-streaming.spec.ts      # SSE 事件流验证
├── memory-library.spec.ts      # 记忆图书馆管理
├── terminal-agent.spec.ts      # 终端代理面板
└── token-piechart.spec.ts      # Token 饼图组件
```

### 5.3 测试场景

**chat-streaming.spec.ts:**
- 发送消息 → 接收 delta 帧 → 文本逐字渲染
- 接收 thinking 帧 → 折叠动画显示
- 接收 context 帧 → Token 饼图更新
- 接收 done 帧 → 消息标记完成
- 取消请求 → cancelled 帧

**memory-library.spec.ts:**
- 管理台图书列表 → 创建图书 → 添加章节
- 搜索 → 显示 FTS5 结果
- Tag 树导航 → 逐层展开

**terminal-agent.spec.ts:**
- 执行 echo 命令 → 实时输出
- 执行 sleep 30 → 终止按钮可用 → 终止后状态更新
- 进程列表 → 显示运行中进程

**token-piechart.spec.ts:**
- 接收 context 帧 → 环形图渲染
- 层详情悬浮 → tooltip 显示
- 高负载 → 颜色变为警告色 (red/orange)
