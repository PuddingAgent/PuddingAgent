## 1. дёҖеҸҘиҜқжҖ»зәІ

**Pudding зҡ„ UI еә”еғҸдёҖдёӘе®үйқҷиҝҗиЎҢзҡ„ AI Runtimeпјҡз”ЁжҲ·еңЁ Agent Console жіЁе…Ҙж„ҸеӣҫпјҢзі»з»ҹеңЁ Runtime Timeline дёӯжЈҖзҙўгҖҒжҖқиҖғгҖҒи°ғз”Ёе·Ҙе…·гҖҒз»„з»ҮдҝЎжҒҜпјҢжңҖз»Ҳзӯ”жЎҲд»ҺиҝҗиЎҢиҝҮзЁӢйҮҢйҖҗжёҗеҮқиҒҡеҮәжқҘгҖӮ**

дёҚжҳҜпјҡ

- иҒҠеӨ©ж°”жіЎдёҖеј еј еј№еҮә
- DOM е…ғзҙ иў« append
- е·Ҙе…·гҖҒжҖқз»ҙгҖҒж¶ҲжҒҜеҗ„иҮӘеј№еҚЎзүҮ
- ж»ЎеұҸйң“иҷ№гҖҒзІ’еӯҗгҖҒжёёжҲҸеҢ–зү№ж•Ҳ

иҖҢжҳҜпјҡ

- дҝЎжҒҜеңЁжөҒеҠЁгҖҒеҲҶеұӮгҖҒзј–з»Ү
- зҠ¶жҖҒжңүе‘јеҗёгҖҒжңүиҠӮеҘҸгҖҒжңүеҒңйЎҝ
- е·Ҙе…·и°ғз”ЁеғҸ Runtime дҝЎеҸ·еңЁзәҝи·Ҝдёӯдј йҖ’
- жңҖз»Ҳзӯ”жЎҲеғҸзҘһз»Ҹе…ғжҝҖжҙ»еҗҺзҡ„ж–Үеӯ—йҖҗжёҗзЁіе®ҡ

е…ій”®иҜҚпјҡ**Fluid / Calm / Rhythm / Neural / Ambient / Cinematic**гҖӮ

## 2. и®ҫи®ЎиҜӯиЁҖж ёеҝғе®ҡд№ү

### 2.1 е‘ҪеҗҚ

е»әи®®е°Ҷж–°и§Ҷи§үиҜӯиЁҖе‘ҪеҗҚдёәпјҡ

**Quiet Runtime Space / е®үйқҷиҝҗиЎҢзҡ„ AI Runtime з©әй—ҙ**

е®ғдёҚжҳҜдј з»ҹиҒҠеӨ©еә”з”ЁпјҢд№ҹдёҚжҳҜдјҒдёҡеҗҺеҸ°еӨ§еұҸпјҢиҖҢжҳҜжң¬ең°з§Ғдәә Agent жӯЈеңЁжҢҒз»ӯж„ҹзҹҘгҖҒжЈҖзҙўгҖҒжҺЁзҗҶгҖҒи°ғз”Ёе·Ҙе…·гҖҒз»„з»Үзӯ”жЎҲзҡ„еҸҜи§ҶеҢ–з©әй—ҙгҖӮ

### 2.2 е…ӯдёӘе…ій”®иҜҚзҡ„ UI и§ЈйҮҠ

| е…ій”®иҜҚ | UI дёӯзҡ„иЎЁиҫҫ | еҝ…йЎ»йҒҝе…Қ |
|---|---|---|
| **Fluid** | дҝЎжҒҜеғҸж°ҙжөҒдёҖж ·иҝӣе…Ҙ TimelineпјҢйҖҗжӯҘжұҮиҒҡдёәз»“жһ„ | еҚЎзүҮзӘҒз„¶еҮәзҺ°гҖҒеёғеұҖи·іеҸҳ |
| **Calm** | дҪҺйҘұе’ҢгҖҒдҪҺеҷӘеЈ°гҖҒжҹ”е’ҢеҸ‘е…үгҖҒз•ҷзҷҪзЁіе®ҡ | ејә bounceгҖҒй«ҳйў‘й—ӘзғҒгҖҒж»ЎеұҸеҠЁж•Ҳ |
| **Rhythm** | Thinking вҶ’ Tool вҶ’ Synthesizing вҶ’ Completed жңүиҠӮеҘҸиө·дјҸ | жүҖжңүеҢәеҹҹеҗҢж—¶еҠЁ |
| **Neural** | з”ЁиҠӮзӮ№гҖҒз»ҶзәҝгҖҒеҫ®е…үиЎЁиҫҫи®°еҝҶ/е·Ҙе…·/дёҠдёӢж–ҮиҝһжҺҘ | иҝҮеәҰе…·иұЎзҡ„еӨ§и„‘гҖҒиөӣеҚҡиЈ…йҘ°е Ҷз Ң |
| **Ambient** | иғҢжҷҜеҸӘжҳҜз©әж°”ж„ҹпјҢдҪҺеҜҶеәҰеҫ®зІ’дёҺйӣҫеҢ–е…ү | зІ’еӯҗжҲҗдёәдё»и§’гҖҒзғҹиҠұзҲҶзӮё |
| **Cinematic** | жҡ—еңәгҖҒжҷҜж·ұгҖҒзј“ж…ўй•ңеӨҙж„ҹгҖҒиҒҡе…үејҸеұӮж¬Ў | з§‘е№»з”өеҪұ HUD е Ҷж»ЎеұҸе№• |

### 2.3 и§Ҷи§үеҺҹеҲҷ

1. **дё»еҶ…е®№ж°ёиҝңдјҳе…Ҳ**пјҡж–Үеӯ—гҖҒе·Ҙе…·з»“жһңгҖҒеҸҜж“ҚдҪңзҠ¶жҖҒзҡ„еҸҜиҜ»жҖ§й«ҳдәҺж°ӣеӣҙгҖӮ
2. **еҠЁж•ҲеҸӘи§ЈйҮҠзҠ¶жҖҒпјҢдёҚеұ•зӨәжҠҖе·§**пјҡз”ЁжҲ·зңӢеҠЁж•ҲжҳҜдёәдәҶзҹҘйҒ“ Agent жӯЈеңЁеҒҡд»Җд№ҲпјҢиҖҢдёҚжҳҜж¬ЈиөҸзү№ж•ҲгҖӮ
3. **зІ’еӯҗж°ёиҝңеңЁиғҢжҷҜеұӮ**пјҡе®ғжҳҜз©әж°”ж„ҹпјҢдёҚжҳҜеҠҹиғҪжҺ§д»¶гҖӮ
4. **еҗҺеҸ°и¶ҠдҪҺйў‘пјҢи¶ҠжҺҘиҝ‘ Ant Design Pro еҺҹз”ҹ**пјҡдёҚиҰҒжҠҠжүҖжңүз®ЎзҗҶйЎөйғҪеҒҡжҲҗз§‘е№»йқўжқҝгҖӮ
5. **Chat жҳҜдё»иҲһеҸ°пјҢConsole жҳҜе·Ҙе…·з®ұ**пјҡзҷ»еҪ•еҗҺй»ҳи®Өиҝӣе…Ҙ `/chat` зҡ„ж–№еҗ‘дҝқжҢҒжӯЈзЎ®гҖӮ

## 5. ж ёеҝғз»„д»¶и®ҫи®Ў

## 9. ж— йҡңзўҚдёҺй•ҝжңҹдҪҝз”ЁиҰҒжұӮ

## 10. иҗҪең°дјҳе…Ҳзә§

### P0пјҡйҳІжӯўж–№еҗ‘и·‘еҒҸ

1. **иЎҘйҪҗз»ҹдёҖ UI и§„иҢғ**
   - е°Ҷжң¬ж–№жЎҲдҪңдёә Runtime Space и®ҫи®ЎеҹәзәҝгҖӮ
   - жҳҺзЎ® tokenгҖҒеҠЁж•Ҳйў„з®—гҖҒйЎөйқўеҲҶзә§гҖӮ

3. **з»ҹдёҖ Runtime Timeline**
   - Thought / Tool / Message з»ҹдёҖдёәиҝһз»ӯдҝЎжҒҜжөҒгҖӮ
   - е·Ҙе…·е’ҢжҖқиҖғй»ҳи®Өж‘ҳиҰҒеҢ–гҖӮ

   - жңҖе®№жҳ“е»әз«Ӣз¬¬дёҖеҚ°иұЎгҖӮ

   - еҠЁж•Ҳзі»з»ҹдёҠзәҝеүҚзҡ„еҝ…иҰҒдҝқжҠӨгҖӮ

- е·Ҙе…·и°ғз”Ёеұ•зӨәе·Ҙе…·еҗҚгҖҒзӣ®зҡ„гҖҒзҠ¶жҖҒе’Ңз®Җзҹӯз»“жһңгҖӮ

### 11.4 жңҖз»Ҳеӣһзӯ”зӘҒеҮә

- жңҖз»Ҳеӣһзӯ”и§Ҷи§үжқғйҮҚй«ҳдәҺжү§иЎҢиҝҮзЁӢгҖӮ

## 15. жңҖз»Ҳзӣ®ж Үз”»йқў

з”ЁжҲ·зңӢеҲ°зҡ„дёҚжҳҜдёҖдёӘвҖңжӯЈеңЁжү“еӯ—зҡ„иҒҠеӨ©жңәеҷЁдәәвҖқпјҢиҖҢжҳҜпјҡ

```text
дёҖдёӘе®үйқҷиҝҗиЎҢзҡ„жң¬ең° AI зі»з»ҹгҖӮ

е®ғжҺҘж”¶дҪ зҡ„ж„ҸеӣҫпјҢ
еңЁдёҠдёӢж–ҮгҖҒи®°еҝҶе’Ңе·Ҙе…·д№Ӣй—ҙжөҒеҠЁпјҢ
жҠҠзҰ»ж•ЈдҝЎжҒҜз»„з»ҮжҲҗз»“жһ„пјҢ
жңҖеҗҺжҠҠзӯ”жЎҲжё©е’ҢгҖҒзЁіе®ҡең°е‘ҲзҺ°еңЁдҪ йқўеүҚгҖӮ
```
