# V1 落地与验收

> **2026-05-02**：V1 目标从分布式平台改为单进程 Agent。

## V1 目标

一个可以双击运行的 Pudding Agent 可执行程序。

## 第一条链路

```
浏览器 → localhost:8080（内嵌 Web UI）
       → Controller 模块（同进程）
       → Runtime 模块（同进程）
       → LLM API → 真实回复
```

## 验收标准

- [ ] 单个可执行文件，双击启动
- [ ] 浏览器打开 http://localhost:8080 可见对话界面
- [ ] 能发送消息并获得 LLM 真实回复
- [ ] 对话历史持久化到 SQLite
- [ ] 两个 Agent 进程可互相发现（P2P）
- [ ] 可通过 Docker 单容器部署

## 废弃

- ❌ Controller/Runtime 分进程部署
- ❌ Nginx/PostgreSQL/Redis/RabbitMQ/MinIO
- ❌ Docker Compose 多容器编排
- ❌ Gateway Adapter Plugin
- ❌ 审批链（V2）
