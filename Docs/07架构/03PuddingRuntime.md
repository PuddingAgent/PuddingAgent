# Agent Runtime（进程内模块）

> **2026-05-02 简化**：Runtime 不再是独立进程，而是 Pudding Agent 内的一个模块。1 进程 = 1 Agent = 1 Runtime 实例。

## 定位

Runtime 模块负责 Agent 的实际执行：LLM 对话、工具调用、记忆管理、Skill 装配。

## 核心能力

- LLM 对话循环（多轮会话、工具调用）
- 记忆引擎（会话记忆、长期记忆召回与写回）
- Skill/MCP 装配与执行
- 工具进程内直接执行

## 数据存储

所有数据存储在进程本地的 SQLite 数据库中：
- 会话历史
- 记忆条目
- 配置与偏好
- 审计日志

不依赖外部数据库服务。
