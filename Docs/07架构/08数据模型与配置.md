# 数据模型

> **2026-05-02**：数据存储从 PostgreSQL + Redis 简化为 SQLite 单文件数据库。

## 核心实体（SQLite）

- Agent 配置：LLM 端点、Key、模型、系统提示词
- Session：对话会话
- Message：对话消息历史
- Memory：长期记忆条目
- PeerNode：已知的对等 Agent 节点
- AuditLog：审计日志

## 配置

- SQLite 数据库文件：`data/pudding.db`（自动创建）
- 配置文件：`appsettings.json`（可嵌入程序集或外置）
- 环境变量：`LLM_API_KEY`、`LLM_ENDPOINT`、`LLM_MODEL`

## 废弃的实体

原 PostgreSQL 中的 Team、Workflow、KnowledgeBase、LlmProvider/Model/Quota、Channel、Member 等实体废弃。
