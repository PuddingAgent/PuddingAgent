# 数据模型

> **2026-05-02**：数据存储从 PostgreSQL + Redis 简化为 SQLite 单文件数据库。

## 核心实体（SQLite）

- Agent 配置：自包含实例 manifest、Markdown 角色文件和 provider/model 引用；
  全局模板仅在创建时复制，运行时不继承或回查模板
- Session：对话会话
- Message：对话消息历史
- Memory：长期记忆条目
- PeerNode：已知的对等 Agent 节点
- AuditLog：审计日志

## 配置

- SQLite 数据库文件：`data/pudding.db`（自动创建）
- 配置文件：`appsettings.json`（可嵌入程序集或外置）
- LLM Provider/Model 配置：`data/config/llm.providers.json`，这是运行时唯一来源
- 模型上下文容量：运行时只从 `llm.providers.json` 的 Model 配置解析；Agent manifest、
  Agent DTO 和 LLM Binding 不复制 `maxContextTokens`
- 旧 LLM 环境变量回退已废弃

## 废弃的实体

原 PostgreSQL 中的 Team、Workflow、KnowledgeBase、LlmProvider/Model/Quota、Channel、Member 等实体废弃。
